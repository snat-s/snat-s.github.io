Usando Gnome-boxes

¿Qué es gnome-boxes?

gnome-boxes es una de las aplicaciónes de gnome que vienen junto con el desktop enviroment de GNU y sirve como un frontend para QEMU.

Aunque existen muchas alternativas1 y hay gente que considera que no es el mejor software. La característica más importante de boxes es que es muy fácil usar y es bueno para empezar a utilizar máquinas virtuales (VM).

Instalación

  • Fedora: # dnf install gnome-boxes
  • Arch: # apt install gnome-boxes
  • Debian/Ubuntu: # pacman -S gnome-boxes

Uso

Puedes descargar .iso de distintas partes de internet o directamente después de hacer click en el botón de + de la esquina superior derecha tienes la opción para descargar un sistema operativo.

Después tienes varias opciónes para poder modificar la cantidad de espacio que tiene el sistema y la cantidad de recursos que va a usar2.

Ahora que ya tenemos la iso y los parámetros configurados, podemos proceder a la instalación del sistema operativo. Que de hecho es exactamente la misma que tendrías si fuera un sistema real.

Listo, al final de la instalación lo único que tenemos que hacer es reiniciar la máquina virtual y terminamos.

Más funcionalidad

Express install

Express install es un modo de instalación de un sistema operativo que automáticamente instala la máquina virtual por ti y que configura todo.

Si quieres probar un sistema rápidamente es una de las formas más fáciles que existen para probar otra cosa.
